reduxbase
=====================

This is a copy of Dan Abramovs [ react-hot-boilerplate](https://github.com/gaearon/react-hot-boilerplate)

There is an example of a stateless component a la react 0.14 in there too

### Dependencies

* React
* Webpack
* [webpack-dev-server](https://github.com/webpack/webpack-dev-server)
* [babel-loader](https://github.com/babel/babel-loader)
* [react-hot-loader](https://github.com/gaearon/react-hot-loader)
